=== Migration

You need to update your Gradle dependency declaration from "care.data4life.hc-fhir-sdk-java:hc-fhir-sdk-java:{version}" to:

[source, gradle]
----
dependencies {
    implementation("care.data4life.hc-fhir-sdk-java:fhir-java:$version")
}
----
